About these documents¶

These documents are generated from reStructuredText sources by Sphinx, a document processor specifically written for the official Python documentation.

In the online version of these documents, you can submit comments and suggest changes directly on the documentation pages.

Development of the documentation takes place on the users mailing list. We’re always looking for volunteers wanting to help with the docs, so feel free to send a mail there!

See Reporting Bugs in PyAMF for information how to report bugs in PyAMF.
