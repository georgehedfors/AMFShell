Module Hierarchy

  • cpyamf: Python C-extensions for PyAMF.

  • pyamf: PyAMF provides Action Message Format (AMF) support for Python that is compatible with the Adobe Flash Player.
    • pyamf.adapters: The adapter package provides additional functionality for other Python packages.
      • pyamf.adapters._array: `array` adapter module.
      • pyamf.adapters._collections: collections adapter module.
      • pyamf.adapters._decimal: Adapter for the decimal module.

      • pyamf.adapters._django_db_models_fields: django.db.models.fields adapter module.
      • pyamf.adapters._django_db_models_query: Django query adapter module.
      • pyamf.adapters._django_utils_translation: django.utils.translation adapter module.
      • pyamf.adapters._google_appengine_ext_db: Google App Engine adapter module.
      • pyamf.adapters._sets: Adapter for the stdlib sets module.
      • pyamf.adapters._sqlalchemy_orm: SQLAlchemy adapter module.
      • pyamf.adapters._sqlalchemy_orm_collections: SQLAlchemy adapter module.
      • pyamf.adapters.util: Useful helpers for adapters.
    • pyamf.amf0: AMF0 implementation.
    • pyamf.amf3: AMF3 implementation.
    • pyamf.flex: Compatibility classes/functions for Flex.
      • pyamf.flex.data: Flex Data Management Service implementation.
      • pyamf.flex.messaging: Flex Messaging implementation.
    • pyamf.remoting: AMF Remoting support.
      • pyamf.remoting.amf0: AMF0 Remoting support.
      • pyamf.remoting.amf3: AMF3 RemoteObject support.
      • pyamf.remoting.client: Remoting client implementation.
      • pyamf.remoting.gateway: Remoting server implementations.
        • pyamf.remoting.gateway.django: Gateway for the Django framework.
        • pyamf.remoting.gateway.google: Gateway for Google App Engine.
        • pyamf.remoting.gateway.twisted: Twisted server implementation.
        • pyamf.remoting.gateway.wsgi: WSGI server implementation.
    • pyamf.sol: Local Shared Object implementation.
    • pyamf.util: AMF Utilities.
      • pyamf.util.imports: Tools for doing dynamic imports.
